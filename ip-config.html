Cette adresse IP permettra aux autres appareils sur le même réseau d'accéder à votre application React.

Configuration nécessaire

1

Ouvrez votre fichier package.json et modifiez la ligne "start" comme suit :

"start": "HOST=0.0.0.0 PORT=4100 react-scripts start"
2

Redémarrez votre serveur de développement :

npm start
3

Sur les autres appareils, ouvrez un navigateur et accédez à :

http://[votre-ip]:4100

Notes importantes

Firewall : Assurez-vous que votre firewall autorise les connexions entrantes sur le port 4100.

Réseau : Les appareils doivent être connectés au même réseau local (Wi-Fi ou Ethernet).

Configuration alternative : Vous pouvez aussi créer un fichier .env à la racine de votre projet avec :

HOST=0.0.0.0
PORT=4100
